## PHASE 4: Google Search Ads (45 mins)

- [ ] In Google Ads, create new campaign: **Search**
- [ ] Campaign name: "Saint & Story - Removals Search"
- [ ] Set locations: United Kingdom
- [ ] Set daily budget: £10
- [ ] Create ad group: "High-Intent Removals"
- [ ] Add keywords (phrase match):
  - [ ] "[city] removals" (for each city)
  - [ ] "house removals near me"
  - [ ] "same-day removals"
  - [ ] "moving company [city]"
  - [ ] "office removals [city]"

- [ ] Create ad copy:
  - [ ] Headline 1: "Same-Day Removals in London"
  - [ ] Headline 2: "Fixed Price. Verified Drivers."
  - [ ] Headline 3: "Available 7AM-10PM"
  - [ ] Description 1: "Book a removal in minutes..."

- [ ] Set up conversion tracking:

  - [ ] Add conversion: "Removal Inquiry"
  - [ ] Conversion URL: https://saintandstoryltd.co.uk/api/leads
  - [ ] Conversion value: £30

- [ ] Check campaign status: **ACTIVE**

**Status:** Search ads live. Users searching "[city] removals" will see your ads.
