# Language Guide: From Corporate Jargon to Human Language

**Core Principle:** Every label, every metric, every section heading must be understandable to an operator WITHOUT corporate translation.

## Translation Map

### Asset/Resource Language
| Corporate | Human |
|-----------|-------|
| Recurring Revenue Assets | Standing Orders / Recurring Business |
| Asset Health Summary | Status |
| Total Assets | Total Programs |
| Healthy Assets | Working |
| Blocked Assets | Blocked |
| Asset Pipeline | Flow |

### Coverage Language
| Corporate | Human |
|-----------|-------|
| Market Coverage | What We're Searching |
| Coverage Status | Search Status |
| Discovery Velocity | Businesses Found |
| Active Industries | Categories We're Looking For |
| Target Market Universe | Markets Available |

### Activity Language
| Corporate | Human |
|-----------|-------|
| Last Activity | Started |
| Revenue Potential | Expected Jobs |
| Background Operations | What's Running |
| Asset Health Indicators | How We're Doing |

### Action Language
| Corporate | Human |
|-----------|-------|
| Next Action | What to Do |
| Attention Required | Needs Attention |
| Blockers Requiring Attention | What's Blocked |

### Intelligence Language
| Corporate | Human |
|-----------|-------|
| Commercial Interest | They're Interested |
| Follow-Up Required | Need to Follow Up |
| High Value Actions | Next Steps to Revenue |

### Movement Language
| Corporate | Human |
|-----------|-------|
| Configured | Set Up |
| Active | Running |
| Producing Jobs | Generating Jobs |

## Application Rule

Before writing ANY label on ANY screen:
1. Ask: "Could an operator understand this without a glossary?"
2. If no: Translate to human language
3. If yes: Use it

## Examples

❌ "Asset Pipeline": Are these pipes made of assets?
✅ "Flow": Clear - movement through stages

❌ "Market Coverage Status": Coverage of what? What status?
✅ "Search Status": Clear - what's our search doing right now

❌ "Blockers Requiring Attention": Vague. What kind of blockers?
✅ "What's Blocked": Clear - blocked programs/standing orders

## Future Changes

As you build new screens:
- Pipeline metrics → "How opportunities are moving"
- Discovery page → "What we're finding"
- Revenue Engine → "Where money is getting stuck"
- Analytics → "What's improving"

Every section should answer ONE business question in human language.
